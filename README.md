# Presença S10
## Antigo Plaquinha S10

O Presença S10 é um sistema que detecta a presença de pessoas e o estado de travamento da porta do Laboratório de Automação e Sistemas Embarcados, localizado na sala S10 do LABOTEC 3, na UEFS, e disponibliza essa informação para que as pessoas possam saber de antemão sobre a possibilidade de acesso ao laboratório.
